# student_management
The Student Management System is a web application built with PHP, HTML, CSS, JavaScript, and SQL, aimed at efficiently managing student data. The system allows administrators to add, edit, view, and delete student information, storing all records in an SQL database. Key functionalities include form-based data entry, real-time validation for email and phone number fields, and a tabular view of the student list. This project emphasizes user-friendly interactions, data integrity, and secure storage of student details, such as name, email, phone number, and address, facilitating easy management and retrieval of student records.
